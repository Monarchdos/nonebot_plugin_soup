## 💿 安装

**使用 nb-cli 安装**  
在 nonebot2 项目的根目录下打开命令行, 输入以下指令即可安装  

```bash
nb plugin install nonebot-plugin-soup
```

**使用 pip 安装**  
```bash
pip install nonebot-plugin-soup
```

打开 nonebot2 项目的 `bot.py` 文件, 在其中写入
```python
nonebot.load_plugin('nonebot_plugin_soup')
```
